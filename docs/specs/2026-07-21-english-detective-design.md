# 英语侦探社 — 产品设计文档

> 面向中国小学三四年级学生（8-10岁）的 iPad 英语学习工具
>
> 设计日期：2026-07-21
>
> 状态：设计定稿，待进入 prototype 阶段

---

## 一、产品定位

一款以"侦探破案"为核心叙事的 iPad 英语学习 App。将听说读写包装为破案流程，让小朋友在沉浸式故事中完成英语学习，而非传统的"学-练-测"模式。

**核心隐喻**：每个教材单元 = 一个案件。你不是在学英语，你是在当侦探。

## 二、目标用户

- **主要用户**：小学三四年级学生（8-10岁）
- **使用设备**：iPad（横屏为主）
- **使用场景**：家庭日常学习，配合学校英语课程进度
- **次要用户**：家长（查看学习报告、管理设置）

## 三、核心功能

### 3.1 教材同步

- 支持切换不同版本教材（人教版 PEP、外研版等）
- 每个教材单元映射为一个"案件"
- 按教材顺序解锁案件

### 3.2 五步破案学习流程

| 步骤 | 侦探叙事 | 学习内容 | 关键交互 |
|------|---------|---------|---------|
| 1. 接案 | 接收密函，了解案情 | 浏览本课单词（卡片：图+拼写+发音+例句） | 火漆融化动画 → 打字机出标题 → 探长登场 |
| 2. 搜证 | 案发现场调查 | 听音选词、看图选词、拼写填空 | 手电筒探照 → 放大镜检视 → 说出单词 → 证物飞入证物袋 |
| 3. 审讯 | 与证人/嫌疑人对话 | 跟读句子，逐词发音评分 | 角色表情动画 → 声波纹跳动 → 逐词反馈（绿/黄/红） |
| 4. 结案报告 | 撰写并提交案件报告 | 拖词填空（拼写测验） | 打字机滚纸 → 单词磁吸入位 → "CASE SOLVED"印章砸下 |
| 5. 庆祝 | 案件解决，获得奖励 | 单元通关结算 | 烟花粒子 + 徽章旋转浮现 + "叮"飞入收藏册 |

### 3.3 每日任务制

- 每日刷新三项任务：学习新单词、复习冷案词、跟读句子
- 每项显示进度条，完成获得经验和金币奖励
- 引导小朋友均衡练习听说读写

### 3.4 跟读与发音评估

- **逐词纠音**：句子中每个单词独立评分
- 标准（绿色弹跳）、可接受（黄色）、需练习（红色抖动）
- 声波纹实时跳动 + 角色表情联动反馈
- 证人对话模式：发音标准 → 证人给更多线索
- 嫌疑人审问模式：纠正嫌疑人的英语错误 → 嫌疑人"崩溃"

### 3.5 易错词复习（冷案系统）

- **自动追踪**：答错/读错的单词自动进入"冷案档案室"
- **艾宾浩斯悬赏令**：按遗忘曲线自动计算复习时机，紧急的贴"红色悬赏令"
- **手动收藏**：小朋友也可以主动加星标（"我的收藏"）
- **每周越狱 Boss 战**：所有待复习冷案词集体"越狱"，一次性全部追捕
- **吹灰动画**：打开冷案卷宗时，吹掉灰尘（对麦克风吹气或点击）

### 3.6 游戏化系统

- **侦探等级**：见习 → 初级 → 资深 → 传奇侦探（经验值升级）
- **徽章收集册**：每个结案的案件获得一枚独特徽章，集齐套装解锁特殊奖励
- **装备收集**：完成学习获得装备碎片（"拼写之剑""发音之盾"等）
- **金币商城**：装饰角色、解锁特殊道具

### 3.7 家长功能

- **家长看板**（简单密码保护）：
  - 每日学习时长、完成任务统计
  - 易错词数量、跟读成绩趋势
  - 学习天数连续记录
- **家长控制**：
  - 每日使用时长上限
  - 调整任务量
  - 管理教材选择

## 四、信息架构

```
🏢 侦探事务所（主界面）
├── 📁 受理新案件 → 接案 → 搜证 → 审讯 → 结案报告 → 庆祝
├── 📂 冷案档案室 → 悬赏令列表 → 逐词复习 → 越狱Boss战
├── 🏆 徽章收藏册 → 已收集 / 未解锁
├── 📚 教材选择 → 教材列表 → 单元/案件列表
└── ⚙️ 设置（含家长入口）
```

## 五、视觉设计

### 5.1 设计语言：暖调案卷风

以"侦探手账 / 旧案卷"为美学参考，温暖、友好、清晰。

### 5.2 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 奶油背景 | `#FFFBEB` | 全局底色，模拟旧案卷纸张 |
| 深棕墨水 | `#44403C` | 主文字色，钢笔墨水质感 |
| 琥珀金 | `#D97706` | 重点/奖励/悬赏令/徽章 |
| 松石绿 | `#65A30D` | 完成状态/正确反馈 |
| 暖红 | `#DC2626` | 紧迫提示/错误（克制使用） |
| 白色卡片 | `#FFFFFF` | 卡片底色，在奶油背景上自然浮起 |
| 浅米色 | `#F5F0E8` | 次级背景/分隔区 |
| 边框 | `#E7E0D5` | 卡片边框/分割线 |

### 5.3 字体

- 标题：Fredoka（圆润、友好、卡通感）
- 正文：Nunito（圆角无衬线、清晰舒适）
- 备选：系统原生字体（确保 iPad 渲染效果）

### 5.4 设计原则

- **暖但不幼稚**：圆角 12-18px 保持友好，但不使用荧光色和 Comic Sans
- **清晰至上**：所有文字在奶油底上 4.5:1+ 对比度
- **状态色克制**：只用绿/琥珀/红三种状态色
- **白色卡片分层**：在奶油背景上自然浮起，不需要厚边框和重阴影
- **童趣不幼稚**：小朋友值得被认真对待，设计品质不能因为是"儿童 App"而打折

## 六、动画语言

贯穿全 App 的四个动画原则：

| 原则 | 说明 |
|------|------|
| 🫧 弹性 | 所有弹出、切换用 spring 弹性曲线，按压沉降到 0.97，让界面"有生命" |
| ✨ 粒子 | 每完成动作：星星、金币、纸屑飞出，即时满足感 |
| 🔊 音画同步 | 盖章"啪"、打字"咔"、星星"叮"、火漆融化"啵" |
| 😄 角色表现力 | 每个 NPC 有性格，夸张表情 + 肢体动作，卡通片级别的表演 |

关键动画节点：
- 接案：密函飞入 → 火漆融化 → 打字机效果（约 6 秒）
- 搜证：手电筒光圈跟随手指 → 放大镜弹出 → 证物缩放飞入袋子
- 审讯：声波纹实时跳动 → 逐词反馈条（绿弹跳/黄摆动/红抖动）→ 证人表情联动
- 结案：印章从天砸下 → "CASE SOLVED" 红色字样 → 纸页四角卷起
- 庆祝：全屏纸屑 + 烟花粒子 → 徽章旋转浮现 → "叮"飞入收藏册
- 冷案：吹灰尘粒散开 → 悬赏令脉冲发光 → 越狱纸人满屏逃跑 → 手铐锁住

## 七、技术考量（待定）

以下在 prototype 阶段进一步评估：

- **语音评测 SDK**：逐词纠音需要儿童语音识别引擎（需适配中国小朋友的英语发音特征）
- **平台**：原生 SwiftUI（首选，iPad 性能最优）vs Flutter（跨平台备选）
- **离线能力**：核心学习内容应支持离线使用
- **数据存储**：学习记录、错词本、进度等本地优先，可选 iCloud 同步

## 八、非功能需求

- 小朋友可以独立完成每日任务（不需要家长全程陪同）
- 所有动画必须可中断（用户点击/手势立即响应）
- 支持 Reduced Motion（减少动效幅度）
- 触摸目标 ≥ 44pt（iPad HIG 标准）
- 首次使用有简单的引导流程（可跳过）

## 九、未来扩展（暂不纳入 MVP）

- 班级排行榜 / 好友对战（社交维度）
- AR 搜证模式（iPad 摄像头找现实中的英文单词）
- 英语绘本阅读模块
- AI 对话练习（自由对话而非固定剧本）
- 多语言版本
